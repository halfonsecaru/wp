Overview

Welcome to the overview panel. This tab demonstrates the basic horizontal layout with smooth slider animation.

Switch between tabs to see the animated underline slider move with a satisfying elastic effect.

Details

Here you can find all the detailed information about the component.

  • Supports keyboard navigation (Arrow keys, Home, End)
  • Touch swipe support for mobile devices
  • ARIA roles and attributes for accessibility
  • Scrollable header when tabs overflow

Configuration

Pass a config input object or use individual inputs to customize the tabs container.

<alf-tabs-container orientation="horizontal" keyboardActivation="automatic">
  <alf-tab label="Tab 1">...</alf-tab>
</alf-tabs-container>
