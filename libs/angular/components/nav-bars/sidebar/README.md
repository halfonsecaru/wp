# SidebarComponent

Componente de barra lateral para navegación principal (desktop/tablet).

- Muestra enlaces de navegación y se adapta al tamaño de pantalla.
- Incluye animaciones de apertura/cierre y hover.
- Usa el estado global (`UiStateService`) para expandir/colapsar.

## Uso
```html
<app-sidebar></app-sidebar>
```
