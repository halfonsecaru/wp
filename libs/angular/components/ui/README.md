# UiStateService

Servicio global de estado de UI para la aplicación Angular.

- Gestiona el estado de visibilidad del menú lateral, menú móvil, etc.
- Proporciona señales reactivas para componentes.
- Permite centralizar la lógica de UI y facilitar la comunicación entre componentes.

## Uso

```typescript
constructor(public ui: UiStateService) {}
```
